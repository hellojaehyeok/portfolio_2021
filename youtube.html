YouTube Data API와 React Hook을 활용하여 만든 유튜브 웹페이지입니다.
메인 페이지에는 검색창과 함께 인기 동영상을 보여주었습니다. 검색 시에는
24개의 비디오 리스트들을 보여주며 클릭을 할 경우 해당 아이템의 정보를 받아와
동영상과 정보들을 보여주었습니다. React Router를 통해 페이지 간 이동을
할 수 있게 하였으며 PostCSS를 통해 크로스 브라우징이 가능하도록 만들었습니다.
YouTube에서 정보를 받아올 때는 Postman을 활용하여 fetch를 손쉽게 받아왔고,
그 외에도 axios를 활용하여 정보를 받아오기도 하였습니다.
Github 배포 시 React Router History가 지원되지 않아 Netlify로 배포하였습니다.
