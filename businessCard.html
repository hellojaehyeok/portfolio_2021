Firebase를 이용하여 만든 명함 제작 웹페이지입니다.
백엔드 개발자 없이 구글에서 제공하는 Firebase API를 통해 사용자의 정보를
서버에 저장합니다. 메인 페이지에서는 구글과 깃허브로 로그인 가능하며
새로 고침, 재접속을 하여도 데이터는 아이디별로 남아있습니다.
Cloudinary를 사용하여 이미지를 웹사이트에 업데이트하고 지정된 이미지 형식으로
변환하여 저장합니다. PostCSS로 크로스 브라우징이 가능하며 React Router를
활용하여 웹페이지 간 이동이 되도록 하였습니다. 디자인 같은 경우 깔끔한 UI를 위하여
뉴모피즘컨셉으로 잡고 나갔고 최근 유행하는 border-radius를 적용하였습니다.
